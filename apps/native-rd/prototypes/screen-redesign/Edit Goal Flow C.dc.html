How it differs
• Reached from the pencil in Focus Mode — no wizard, no progress bar.
• The same step list as create-screen 3, but persistent: drag ≡ to reorder, tap to rename, × to remove.
• Title is editable inline; Delete goal lives in the ⋯ menu, off the main surface.
• Done returns you to Focus — no “Start working” commitment.
✓ Quick add
From the name screen, confident users tap Quick add to skip the guide and land in the full list (this edit view) — build every step at once. The guided path stays the default.
