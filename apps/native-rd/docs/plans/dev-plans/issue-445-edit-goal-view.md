## Objective

Ship a pure, prop-driven `EditGoalView` component (+ its `EditGoalStepRow` and `EditGoalOverflowMenu` parts) implementing the App Shell `edit` route: an editable goal-title card, drag-reorderable step rows (title + always-present planned-evidence chip + optional display-only date/dependency chips), an inline "Add step..." row, a dates/deps info banner, and a "Done" footer — with "Delete goal" demoted into a `⋯` overflow menu instead of a loud destructive button. Component + Storybook only; no screen wiring (that's a future `[Integrate]` issue, symmetric with how #410/#412 shipped Badge Detail's header/menu ahead of #380).
