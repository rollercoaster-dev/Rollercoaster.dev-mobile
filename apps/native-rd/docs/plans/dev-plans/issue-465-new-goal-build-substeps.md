## Objective

Add a one-level "sub-steps" tier to the New Goal wizard's build step (step 3), reusing `EditGoalSubStepRow` (and its `EditGoalSubStep` type) from `#445` unmodified â€” no fork. A build row with sub-steps renders the same indented mint-railed block of sub-rows already shipped in `EditGoalView`; a row with none shows a "break into sub-steps" prompt. Wiring (rename, evidence, delete) piggybacks on the wizard's existing id-keyed shared state (`editingBuildStepId`, `openBuildStepEvidenceId`, `pendingDeleteBuildStepId`) from #482/#464 rather than adding parallel prop sets, since ids are unique across build steps and their sub-steps. Stays Storybook-only (no navigation/persistence; `[Integrate]` #444 owns that), matching every prior New Goal wizard slice's shape.
